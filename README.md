1# 🎯 FaceID — Binary Face Recognition Web App

A Flask-based web application that performs real-time face classification (ME vs NOT ME) using:
- **Conventional ML** (scikit-learn: Logistic Regression / kNN / Decision Tree)  
- **Teachable Machine** (Google's browser-based TensorFlow.js model)

---

## 🚀 Setup & Run (Step by Step)

### Step 1 — Clone & Install

```bash
git clone https://github.com/YOUR_USERNAME/face-recognition-app.git
cd face-recognition-app

python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

pip install -r requirements.txt
```

### Step 2 — Collect Your Dataset

You need at least **100 images per class** for good accuracy.

```bash
# Capture YOUR face images (press SPACE or 'A' for auto-capture)
python collect_dataset.py --label ME --count 100

# Then ask a friend, or use images from internet for NOT_ME
python collect_dataset.py --label NOT_ME --count 100
```

**Tips for good data:**
- Vary lighting: bright, dim, side-lit
- Vary angles: straight, left, right, up, down
- Vary distances: close, medium, far
- Vary expressions: neutral, smiling, serious

### Step 3 — Train the sklearn Model

```bash
python train_model.py
```

This will:
- Load all images from `dataset/ME/` and `dataset/NOT_ME/`
- Train Logistic Regression, kNN, and Decision Tree
- Save the best model to `models/sklearn_model.pkl`
- Show a comparison chart and confusion matrix

### Step 4 — Train Teachable Machine Model

1. Go to: https://teachablemachine.withgoogle.com/
2. Click **"Get Started"** → **"Image Project"** → **"Standard image model"**
3. Rename Class 1 → `ME`, Class 2 → `NOT_ME`
4. Upload or webcam-capture your images into each class
5. Click **"Train Model"**
6. Click **"Export Model"** → **"TensorFlow.js"** tab → **"Upload my model"**
7. Copy the shareable link (e.g., `https://teachablemachine.withgoogle.com/models/XXXXX/`)

### Step 5 — Run the Flask App

```bash
python app.py
```

Open: http://localhost:5000

In the web app:
- Paste your Teachable Machine URL in the bottom-right panel and click **"Load TM Model"**
- Click **"Start Camera"** → **"Capture & Predict"** to classify your face
- Or switch to **"Upload Image"** to test with a file

## 🌐 Deploy to Render.com (Free Hosting)

1. Push your code to GitHub (see GitHub section below)
2. Go to https://render.com → New → Web Service
3. Connect your GitHub repo
4. Set:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app --bind 0.0.0.0:$PORT`
   - **Environment:** Python 3.11
5. Click **Deploy**

> ⚠️ **Note:** The sklearn model file (`models/sklearn_model.pkl`) must be committed to GitHub for Render to have it. Either commit it directly, or retrain on the server.

### Limitations Observed

1. **sklearn model:** Relies on Haar cascade face detection. Fails in very low light or extreme angles where no face is detected, falling back to full-image features.
2. **Teachable Machine:** Sensitive to lighting changes. High accuracy in controlled conditions but degrades with novel conditions not in training data.
3. **Both systems:** Binary classification is a simplified version of real face recognition. Does not verify identity with high certainty — could be fooled by photos.
